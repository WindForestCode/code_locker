code_locker
=====

Домашнее задание 10

Выполнено как приложение rebabr3.

Был добавлен функционал:

- Новое состояние suspended, в которое замок переходит, если неправильный код введен 3 раза подряд. Все попытки ввести код в состоянии suspended возвращают ошибку. Через 10 секунд (timeout) замок переходит из состояния supsended в locked.
- Возможность изменить секретный код замка, в состоянии open.

Что можно улучшить
-
- Считывать секретный код из конфига;
- Корректно обработать ввод кнопки lock - нажатие считается за часть пароля.
